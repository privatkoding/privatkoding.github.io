Privat Online Backend Intensif dengan Express.js

Kurikulum Express.js (12 Pertemuan)

Total Durasi: 24 jam

  1. Pertemuan 1: Pengenalan Backend dan JavaScript Dasar
    • Apa itu Backend dan Arsitektur Client-Server
    • Variabel, Tipe Data, Operator, Kontrol Alur
    • Looping, Fungsi, dan Scope
    • Praktik: Fungsi manipulasi string dan array
  2. Pertemuan 2: Pengenalan Node.js dan Asynchronous
    • Instalasi Node.js dan npm
    • Modul bawaan Node.js: fs, http, path, os
    • Callback, Promise, Async/Await
    • Praktik: Baca/tulis file dengan fs
  3. Pertemuan 3: Express.js Dasar
    • Instalasi dan pengenalan middleware
    • Membuat server pertama dan routing dasar
    • Struktur folder project
  4. Pertemuan 4: Routing dan Middleware
    • Routing dinamis (param & query)
    • express.json(), urlencoded(), middleware kustom
    • Error handling & praktik autentikasi sederhana
  5. Pertemuan 5: Database dengan Sequelize & MySQL
    • ORM & konfigurasi database
    • Membuat model & koneksi
    • CRUD sederhana
  6. Pertemuan 6: Relasi Database
    • One-to-One, One-to-Many, Many-to-Many
    • Migrasi dan query relasi
    • Praktik: Relasi User & Post
  7. Pertemuan 7: Autentikasi dan Otorisasi
    • JWT, bcrypt, login & register
    • Middleware autentikasi & RBAC
  8. Pertemuan 8: Validasi dan Upload File
    • express-validator / Joi
    • Multer untuk upload avatar
    • Error handling form
  9. Pertemuan 9: Optimasi API dan Caching
    • Redis cache, pagination
    • Praktik caching Redis
  10. Pertemuan 10: Testing API
    • Jest untuk unit test, Supertest untuk integration test
    • Praktik testing API
  11. Pertemuan 11: Deployment dan CI/CD
    • Deploy ke VPS, Docker + PM2
    • Nginx sebagai reverse proxy
  12. Pertemuan 12: Best Practice dan Studi Kasus
    • Struktur folder, logging Winston, security best practices
    • Studi kasus: Todo App dengan JWT & MySQL

Output Akhir:

  • ✅ API lengkap dengan autentikasi dan CRUD
  • ✅ Validasi, upload file, dan caching
  • ✅ Unit test dan deployment production-ready
