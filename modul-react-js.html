Kurikulum React.js (12 Pertemuan)

Total Durasi: 24 jam

  1. Pertemuan 1: HTML, CSS, dan Tailwind CSS
    • Struktur dasar HTML & elemen penting
    • CSS dasar: Box Model, Flexbox, Grid
    • Pengenalan Tailwind CSS
    • Studi Kasus: Landing page sederhana
  2. Pertemuan 2: Setup Proyek React & Tailwind
    • Apa itu React & manfaatnya
    • Setup React dengan Vite
    • Integrasi Tailwind CSS
    • Struktur folder dan komponen awal
  3. Pertemuan 3: JSX, Components, dan Props
    • Penulisan JSX
    • Membuat dan menggabungkan komponen
    • Penggunaan props
    • Studi Kasus: Komponen Card dinamis
  4. Pertemuan 4: State dan Event Handling
    • useState dan perubahan nilai state
    • Event handler & binding
    • Studi Kasus: Counter App
  5. Pertemuan 5: React Hooks Lanjutan
    • useEffect, useContext
    • Side effects dan subscription
    • Studi Kasus: Jam digital dengan useEffect
  6. Pertemuan 6: Fetching API
    • Axios vs Fetch API
    • Loading & error state
    • Studi Kasus: List data pengguna dari API
  7. Pertemuan 7: Navigasi dengan React Router
    • Setup react-router-dom
    • BrowserRouter, Routes, Route
    • Link dan useNavigate
    • Studi Kasus: Halaman Home, About, Contact
  8. Pertemuan 8: State Management dengan Redux Toolkit
    • Instalasi Redux Toolkit
    • Membuat store, slice, selector, dispatcher
    • Studi Kasus: TODO App dengan Redux
  9. Pertemuan 9: Form Handling dan Validasi
    • React Hook Form dan yup/zod
    • Validasi input
    • Studi Kasus: Form login validasi dinamis
  10. Pertemuan 10: Autentikasi dan Proteksi Halaman
    • Mengelola token dengan localStorage
    • Proteksi route dengan middleware
    • Studi Kasus: Halaman profile hanya untuk user login
  11. Pertemuan 11: Deployment React App
    • Build React untuk produksi
    • Deploy ke Netlify dan Vercel
    • Integrasi dengan backend Express
  12. Pertemuan 12: Optimasi dan Performance
    • React.lazy, Suspense, useMemo, useCallback
    • react-window untuk virtualization
    • Lazy loading image

Output Akhir:

  • ✅ Komponen modular & interaktif
  • ✅ Form validasi & autentikasi
  • ✅ Navigasi multi-page dengan React Router
  • ✅ Integrasi API dan deployment ke Netlify/Vercel
  • ✅ Optimasi performa aplikasi
